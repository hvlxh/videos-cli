# videos-cli

An CLI to download videos from various platforms! (Recommended to use Node >17.x.x)

## Supported Platforms
1. YouTube
2. Instagram
3. TikTok

## Installation
1. Run the command `npm i github:hvlxh/videos-cli --global`
2. You have the access to the command `videos`!.
